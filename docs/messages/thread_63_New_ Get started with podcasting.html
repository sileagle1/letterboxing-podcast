CNET is the group that does free classes as a benefit for Best Buy
customers. I buy quite a bit at Best Buy, so it's a natural for me.
The class series are basic but cover every topic: how to set up a
home network, how to transfer all your VHS home movies to DVD's, how
to set up an interactive blog, etc etc. And even though the classes
are basic, the real value is the info relayed through the informal
class discussions between the teacher of the class and the students. I
like how they are set up as if you know NOTHING...but give choices to
those who might already have SOME knowledge.

Now if they would just do more MAC based stuff... :)
