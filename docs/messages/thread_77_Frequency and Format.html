Frequency and Format

From: johnoballou (johnjballou@hotmail.com) | Date: 2007-06-27 18:56:45 UTC
I need some input.

I'm picking up the ball and running with it here for Letterpod. The
old episodes are reposted on iTunes now under simply Letterpod, no
longer "Letterboxing Podcast" though it will show up under searches
for that as well.

Anyway, the point of this post is about future episodes, and the way
we listen to them.

One idea I had was on how frequently we can get out an episode. I was
thinking seasonally. Under this model, we might get 4 a year, and the
episodes would be chock full of features like Orion's Tales from the
Trails, interviews, maybe some clues thrown in. This format is a bit
like a variety show. Drawback, the finished product is rather long to
listen to.

Second idea is to still do all the same things...features, interviews,
stories, etc....but release them as an episode when they are done,
when we get them. We'll still start them with an intro, but maybe it
would be Leterpod: Tales from the Trial by Orion, and that is the
whole episode. This model is shorter, quicker to listen to, and we
can keep them coming more often. The next episode after that might be
Letterpod: Interview with Randy Hall (just making stuff up here).

What do people think? Did I explain these enough?

jackbear
