Gizmo Project

From: johnoballou (johnjballou@hotmail.com) | Date: 2007-08-14 13:18:39 UTC
Wanted to share a cool service with you all called the Gizmo Project.
This is similar to Skype if you know about computer telephony, but
Gizmo has one key advantage for me...you can record a phone conversation.

Gizmo is a free download. What you would need is some microphone on
your computer, whether built in or added on. Then you can call other
computers that have Gizmo and talk for free, or you can call any
telephone in the world (US rates are like $0.01 a minute...pretty
cheap!) So like I said, once you are talking...you can press a record
button and it will record the conversation, and save it on your
desktop as a .WAV file that you can then listen to in Quictime, or
whatever editing software you want to use. I plan to use this to
record interviews with other letterboxers. Wanted to send it along
for you...I like it so far.

Here's the website....
http://gizmoproject.com/
